# Tugas Biodata Web 2 - React

Aplikasi web biodata interaktif yang dibangun menggunakan React, TypeScript, dan Vite. Proyek ini merupakan tugas untuk mata kuliah Pemrograman Web 2, menampilkan profil pribadi, fitur pengecekan zodiak, dan daftar film favorit dengan desain bertema retro 8-bit yang responsif.

## ✨ Fitur Utama
- **Profil & Biodata**: Menampilkan informasi diri dengan efek animasi mengetik (*typing effect*).
- **Cek Zodiak**: Fitur interaktif dengan pop-up *alert* kustom untuk menampilkan zodiak berdasarkan tanggal lahir.
- **Film Favorit**: Menampilkan daftar film aksi favorit dengan fungsionalitas *horizontal scroll* khusus.
- **Tema Retro 8-Bit**: UI bergaya *brutalism* dengan bingkai tebal, *hard-shadow*, dan kursor *pixel art* kustom.
- **Animasi Dinamis**: Transisi masuk, *hover state*, dan pergerakan elemen yang ditenagai oleh pustaka `framer-motion`.

## 💻 Teknologi yang Digunakan
- React (dengan konfigurasi React Compiler)
- Vite
- TypeScript
- Framer Motion
- Vanilla CSS (Flexbox, Grid, Keyframes, Custom Properties)

## 🚀 Panduan Menjalankan Secara Lokal

1. Pastikan komputer Anda telah terinstal Node.js.
2. Lakukan clone pada repositori ini:
   ```bash
   git clone <URL_REPOSITORI_ANDA>
   ```
3. Pindah ke dalam direktori proyek:
   ```bash
   cd tugas-biodata-web-2-react
   ```
4. Instal semua dependensi yang dibutuhkan:
   ```bash
   npm install
   ```
5. Jalankan server *development*:
   ```bash
   npm run dev
   ```
6. Buka `http://localhost:5173` di browser Anda untuk melihat hasilnya.
